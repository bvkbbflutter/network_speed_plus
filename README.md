# network_speed_plus

A Flutter plugin for monitoring real-time upload and download network speed on Android and iOS.

## Features

- ✅ Real-time Upload & Download Monitoring
- ✅ Total Device Traffic Monitoring
- ✅ App-specific Traffic Monitoring
- ✅ Configurable Update Intervals
- ✅ Lightweight & Efficient
- ✅ Android & iOS Support
- ✅ Easy-to-use API
